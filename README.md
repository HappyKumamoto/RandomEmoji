# ランダム Emoji
- 指定された秒数を当てるゲームです。
- リロードする度に当てる「秒数」(5〜10)と、押す標的の「Emoji」(30種以上)は変わります。
- 当てる秒数との誤差により、3段階のリアクションが出現します。<br>
